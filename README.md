# AI Crop Yield Prediction

This project predicts crop yield using Machine Learning.

## SDG Goal
SDG 2: Zero Hunger

## Problem Statement
Develop a Machine Learning model that predicts crop yield based on environmental factors such as rainfall, temperature, humidity, and soil nutrients.

## Technologies
- Python
- Pandas
- Scikit-Learn
- Streamlit

## How to Run

Install dependencies

pip install pandas numpy scikit-learn streamlit joblib

Train Model

python train_model.py

Run Prediction

python predictor.py

Run Web App

streamlit run app.py
